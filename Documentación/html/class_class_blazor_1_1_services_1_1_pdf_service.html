Detailed Description

This service generates a PDF report summarizing the user's nutrition statistics over a specified date range. It uses the QuestPDF library to create a well-formatted document that includes key metrics such as average calories, protein, carbs, and fat consumed, as well as the number of days with menus and how many days the user's goals were met or not met.

byte[] ClassBlazor.Services.PdfService.GenerateStatisticsPdf ( RangeNutritionSummary summary)

Generates a PDF report containing nutrition statistics for the specified date range.

The generated PDF includes information such as the date range, days with menus, days goals were met or not met, and average nutritional values. The format is suitable for sharing or archiving nutrition reports.
